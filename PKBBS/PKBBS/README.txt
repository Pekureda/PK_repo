Compiled Boost 1.78.0 library is not provided.
Version 1.78.0 was used, but it might work with earlier releases of the Boost.

Library and include paths in the MS VS2022 are organized as follows (assuming that the source of this README file is the root):
Boost_1.78.0
	|
	include
		|
		...
	lib
		|
		...

Where include provides headers and lib provides library files.
